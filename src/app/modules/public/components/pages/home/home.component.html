Passport-multer-material

MEAN Stack app with local(+nodemailer mails sent for verification) and google oauth20 authentication methods and adding/editting/removing posts with multer/gridfs uploads. Uses ngrx for state management.

Usage:

  • yarn watch: Compile typescript server(backend) files into js
  • yarn dev:Run the client & server with concurrently
  • yarn server: Run the Express server only
  • ng serve --open Run the client(Angular) only
Server runs on http://localhost:3000 and client on http://localhost:4200. Remember about adding .env files with needed keys and values in root folder. State management in angular is provided with Ngrx.
You can deploy it for example on heroku by connecting with github, eventually you can create build folder(dist) from root folder with: yarn build
